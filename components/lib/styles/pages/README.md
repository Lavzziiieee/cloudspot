# 🌥️ Cloudspot

A simple SaaS boilerplate app like Hubspot built with Next.js + Firebase Auth.

### 🔐 Features
- Email/password login & signup
- Firebase authentication
- Protected dashboard
- Clean layout with basic styling

### ⚙️ Stack
- Next.js
- Firebase
- Vercel (for deployment)
